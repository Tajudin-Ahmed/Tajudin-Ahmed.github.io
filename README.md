# Tajudin Portfolio

A modern portfolio website built with Vite, React, React Router, and AOS animations.

## Tech Stack

- **Vite** - Next generation frontend tooling
- **React 18** - UI library
- **React Router** - Client-side routing
- **React Helmet Async** - Document head management
- **AOS** - Animate on scroll library

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The development server starts at `http://localhost:5173`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
